Notes: Use the config.rb to add more holidays (2016, 2017, etc...) to generate future schedule

Run the program use this command: ruby main.rb

The Commands:
Enter 'Today' to display today's Support Hero.
Enter 'User <name>' (e.g. User Boris) to display user's schedule.
Enter 'Current' to display full schedule for all users in the current month.
Enter 'Mark <date>' (e.g. Mark 2015-4-1) to mark one of their days on duty as undoable.
Enter 'Swap <fromDate> <toDate>' (e.g. Swap 2015-4-1 2015-5-1) to swap user's duty.
Enter 'Exit' to end the program.
